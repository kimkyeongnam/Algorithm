# 빠른 A+B [[link]](https://www.acmicpc.net/problem/15552)
**사용언어: C++14**

![Capture](https://user-images.githubusercontent.com/38516906/65825176-e12bb380-e241-11e9-8561-80a2f33ed8ca.PNG)

```
#include <iostream>

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    
    int T;
    int A,B;
    
    cin>>T;
    for(int i=0; i<T; i++){
        cin>>A>>B;
        cout<<A+B<<"\n";
    }
    
    return 0;
}
```
