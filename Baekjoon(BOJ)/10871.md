# X보다 작은 수 [[link]](https://www.acmicpc.net/problem/10871)
**사용언어: C++14**

![Capture](https://user-images.githubusercontent.com/38516906/65825685-c9efc480-e247-11e9-89d4-7cdb64e08d42.PNG)

```
#include <iostream>

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    
    int N,X;
    int A;
    
    cin>>N>>X;
    for(int i=0; i<N; i++){
        cin>>A;
        if(A<X) cout<<A<<" ";
    }
        
    return 0;
}
```
