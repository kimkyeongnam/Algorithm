# 알파벳 찾기 [[link]](https://www.acmicpc.net/problem/10809)
**사용언어: C++14**

![Capture](https://user-images.githubusercontent.com/38516906/66002044-c7bc7e80-e470-11e9-83c7-9be9d9f04aa4.PNG)

```
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

int main()
{
    string str;
    cin>>str;
    
    for(int i=0; i<26; i++)
        if(str.find('a'+i) != string::npos)
            cout<<str.find('a'+i)<<" ";
        else cout<<"-1 ";
    return 0;
}
```
