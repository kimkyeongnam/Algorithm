# 두 수 비교하기 [[link]](https://www.acmicpc.net/problem/1330)
**사용언어: C++14**

![Capture](https://user-images.githubusercontent.com/38516906/65824129-6ce91400-e231-11e9-9df2-35d4dc417773.PNG)

```
#include <iostream>

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    
    int A,B;
    cin>>A>>B;
    
    if(A==B) cout<<"==";
    else if(A>B) cout<<">";
    else cout<<"<";
    
    return 0;
}
```
