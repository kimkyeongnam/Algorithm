# 피보나치 수 2 [[link]](https://www.acmicpc.net/problem/2748)
**사용언어: C++14**
![캡쳐](https://user-images.githubusercontent.com/38516906/61681303-f3700980-ad47-11e9-9f90-4022a03a65a9.png)

```
#include <iostream>
using namespace std;

int main()
{
    int n;
    cin >> n;
    long *arr = new long[n+1];

    arr[0] = 0;
    arr[1] = 1;

    for (int i = 2; i <= n; i++){
	arr[i] = arr[i - 1] + arr[i - 2];
    }
    cout << arr[n];
    return 0;
}
```
