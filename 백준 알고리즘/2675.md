# 문자열 반복 [[link]](https://www.acmicpc.net/problem/2675)
**사용언어: C++14**

![Capture](https://user-images.githubusercontent.com/38516906/66001355-4e705c00-e46f-11e9-82af-fa867e93ac39.PNG)

```
#include <iostream>
#include <string>

using namespace std;

int main()
{
    int N;
    cin>>N;
    
    int P;
    string str;
    for(int i=0; i<N; i++){
        cin>>P>>str;
        for(int j=0; j<str.size(); j++)
            for(int k=0; k<P; k++)
                cout<<str[j];
        cout<<endl;
    }
    return 0;
}
```
