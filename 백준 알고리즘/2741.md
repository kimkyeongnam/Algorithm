# N 찍기 [[link]](https://www.acmicpc.net/problem/2741)
**사용언어: C**

![Capture](https://user-images.githubusercontent.com/38516906/65825298-5b106c80-e243-11e9-9419-43cdc953ade2.PNG)

```
#include<stdio.h>

int main(void)
{
	int n,i;

	scanf("%d", &n);

	for(i=1; i<=n; i++)
		printf("%d\n", i);

	return 0;
}
```
