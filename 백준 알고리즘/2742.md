# 기찍 N [[link]](https://www.acmicpc.net/problem/2742)
**사용언어: C**

![Capture](https://user-images.githubusercontent.com/38516906/65825350-fb669100-e243-11e9-87c2-1b8154531ea9.PNG)

```
#include<stdio.h>

int main(void)
{
	int n,i;

	scanf("%d", &n);

	for(i=n; i>=1; i--)
		printf("%d\n", i);

	return 0;
}
```
